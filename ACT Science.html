📑 三种文章类型

Data Representation 数据呈现 (约3篇)

包含图表、表格、图形。考察读取、理解、推断数据的能力。

  • 不需要背景知识，所有信息都在图表中
  • 关注坐标轴标签、单位、趋势
  • 注意图例和数据范围

Research Summaries 研究摘要 (约3篇)

描述一个或多个实验/研究。考察实验设计、变量、结果分析。

  • 识别自变量(Independent)和因变量(Dependent)
  • 控制变量(Controlled variables)是什么
  • 比较不同实验组的差异

Conflicting Viewpoints 冲突观点 (1篇)

呈现2-4个科学家/学生的不同假说。考察理解、比较、评估不同观点。

  • 每个观点的核心主张是什么
  • 观点之间的相同点和不同点
  • 哪些证据支持/反对哪个观点

⚡ 时间分配策略

Passage 1-5: 每篇约 5 分钟
Passage 6 (Conflicting): 约 7-8 分钟
检查: 约 2-3 分钟

🎯 核心解题策略

1 先读题，再看文章

ACT科学不需要逐字阅读文章。先看题目问什么，然后带着问题去图表/文章中找答案。

2 关注图表细节

坐标轴标签、单位、刻度、图例、趋势线、数据点。题目经常考这些细节。

3 识别变量关系

正相关(↑↑) | 负相关(↑↓) | 无相关 | 非线性关系。直接在图表中找趋势。

4 排除法

先排除明显错误的选项。ACT科学很多题目可以通过排除2-3个错误选项快速定位答案。

5 不要过度推理

答案必须基于文章/图表提供的信息。不要带入课外知识（即使你知道更多）。

6 跳过难题，标记回头做

35分钟很紧张。如果一道题超过1分钟没思路，先标记（在答题卡上做个记号），做完其他题再回来。

🧬 生物 (Biology) 常考知识点

Cell Biology Genetics Ecology Evolution Human Body

1. 细胞生物学

  • 细胞膜：选择性透过，控制物质进出
  • 细胞器：线粒体(能量)、叶绿体(光合作用)、核糖体(蛋白质合成)
  • 细胞分裂：有丝分裂(Mitosis) vs 减数分裂(Meiosis)
  • 光合作用：6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂ (需要光、叶绿素)
  • 细胞呼吸：C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + ATP

2. 遗传学

  • DNA结构：双螺旋，碱基配对 A-T, G-C
  • 基因型 vs 表现型：Genotype是基因组合，Phenotype是表现出来的性状
  • 显性(Dominant) vs 隐性(Recessive)：大写字母表示显性，小写隐性
  • Punnett Square：用于预测后代基因型概率
  • 突变(Mutation)：DNA序列的改变，可能有害、有益或中性

3. 生态学

  • 食物链/食物网：生产者(植物) → 初级消费者(食草) → 次级消费者(食肉)
  • 能量金字塔：每级传递约10%能量，其余以热散失
  • 种群增长：指数增长(无限资源) vs 逻辑增长(S型曲线，有限资源)
  • 竞争、捕食、共生：种间关系类型
  • 生态位(Niche)：生物在生态系统中的角色和位置

4. 进化论

  • 自然选择：适者生存，有利变异更可能传递
  • 适应性(Adaptation)：生物特征与环境匹配
  • 遗传变异：自然选择的基础
  • 物种形成(Speciation)：新物种产生的过程

⚗️ 化学 (Chemistry) 常考知识点

Atomic Structure Chemical Reactions Stoichiometry Solutions Acids & Bases

1. 原子结构

  • 质子(Proton)：带正电，决定元素种类（原子序数）
  • 中子(Neutron)：不带电，与质子共同构成原子核
  • 电子(Electron)：带负电，绕核运动，参与化学键形成
  • 同位素(Isotope)：质子数相同，中子数不同
  • 离子(Ion)：失去/获得电子的原子，阳离子(+)，阴离子(-)

2. 化学键

  • 离子键(Ionic)：金属+非金属，电子转移，如 NaCl
  • 共价键(Covalent)：非金属+非金属，电子共享，如 H₂O
  • 金属键(Metallic)：金属原子间，自由电子"海"
  • 极性(Polarity)：电荷分布不均匀，影响溶解性

3. 化学反应

  • 合成反应：A + B → AB
  • 分解反应：AB → A + B
  • 置换反应：A + BC → AC + B
  • 双置换反应：AB + CD → AD + CB
  • 燃烧反应：燃料 + O₂ → CO₂ + H₂O

4. 溶液与浓度

  • 摩尔浓度(Molarity, M)：mol/L
  • 溶解度(Solubility)：一定温度下，100g溶剂最多溶解的溶质质量
  • 饱和(Saturated)：达到最大溶解量
  • 过饱和(Supersaturated)：超过正常溶解度（不稳定）
  • 稀释：M₁V₁ = M₂V₂

5. 酸碱

  • pH 范围：0-14，7为中性，<7酸性，>7碱性
  • pH = -log[H⁺]：每差1个pH单位，[H⁺]差10倍
  • 强酸：HCl, H₂SO₄, HNO₃ — 完全电离
  • 强碱：NaOH, KOH — 完全电离
  • 中和反应：酸 + 碱 → 盐 + 水

🔭 物理 (Physics) 常考知识点

Mechanics Electricity Waves Thermodynamics

1. 力学

  • 速度(Velocity)：位移/时间，矢量（有方向）
  • 加速度(Acceleration)：速度变化率，a = Δv/Δt
  • 牛顿第一定律：惯性，物体保持静止或匀速直线运动
  • 牛顿第二定律：F = ma，力=质量×加速度
  • 牛顿第三定律：作用力与反作用力大小相等、方向相反
  • 重力：F = mg，g ≈ 9.8 m/s² (地球表面)
  • 摩擦力：阻碍相对运动，与正压力成正比

2. 电学

  • 电流(Current, I)：电荷流动速率，单位安培(A)
  • 电压(Voltage, V)：电势差，单位伏特(V)
  • 电阻(Resistance, R)：阻碍电流，单位欧姆(Ω)
  • 欧姆定律：V = IR
  • 串联：R_total = R₁ + R₂ + R₃，电流相同
  • 并联：1/R_total = 1/R₁ + 1/R₂ + 1/R₃，电压相同
  • 电功率：P = VI = I²R = V²/R

3. 波动

  • 波长(Wavelength, λ)：相邻波峰/波谷的距离
  • 频率(Frequency, f)：每秒振动次数，单位赫兹(Hz)
  • 波速：v = f × λ
  • 振幅(Amplitude)：最大位移，决定波的强度/响度
  • 声波：纵波，需要介质，不能在真空中传播
  • 光波：横波，电磁波，可在真空中传播
  • 折射：光从一种介质进入另一种时方向改变，n = c/v

4. 热力学

  • 温度：分子平均动能的量度
  • 热量(Heat)：能量传递形式，单位焦耳(J)或卡路里(cal)
  • 比热容(Specific Heat)：使1g物质升高1°C所需热量
  • 热传递方式：传导(固体)、对流(流体)、辐射(电磁波)
  • 热膨胀：温度升高，大多数物质体积增大
  • 物态变化：熔化、凝固、汽化、液化、升华、凝华

🌍 地球科学 (Earth Science) 常考知识点

Geology Meteorology Oceanography Astronomy

1. 地质学

  • 岩石类型：火成岩(Igneous)、沉积岩(Sedimentary)、变质岩(Metamorphic)
  • 板块构造：地壳分为若干板块，不断运动
  • 地震：板块边界应力释放，震源(Focus)、震中(Epicenter)
  • 火山：岩浆喷出地表，沿板块边界分布
  • 风化作用：物理风化(温度、冰冻)和化学风化(水、酸)
  • 侵蚀与沉积：水、风、冰搬运物质，沉积形成新岩层

2. 气象学

  • 大气层：对流层(Troposphere)、平流层(Stratosphere)、中间层、热层
  • 气压：空气重量产生的压力，高气压→下沉气流→晴天；低气压→上升气流→阴雨
  • 风：从高压吹向低压，受科里奥利力影响偏转
  • 水循环：蒸发 → 凝结 → 降水 → 径流/渗透
  • 锋面：冷锋(冷空气推进)、暖锋(暖空气推进)、静止锋
  • 温室效应：CO₂等气体吸收红外辐射，使地表温度升高

3. 海洋学

  • 洋流：暖流(低纬→高纬)、寒流(高纬→低纬)，影响气候
  • 潮汐：月球引力主导，太阳引力辅助，大潮(Spring tide)、小潮(Neap tide)
  • 盐度：海水含盐量，蒸发强则盐度高，降水/融冰则盐度低
  • 深海热液：海底火山口，高温高压，特殊生态系统

4. 天文学

  • 太阳系：8大行星，按距离太阳：水金地火木土天海
  • 恒星演化：星云 → 原恒星 → 主序星 → 红巨星 → 白矮星/中子星/黑洞
  • 光年(Light-year)：光一年走的距离，约9.46×10¹² km
  • 光谱：恒星成分可通过光谱分析（发射/吸收谱线）
  • 宇宙膨胀：星系相互远离，红移现象证明

📐 核心公式速查

🔭 物理公式

v = d / t   (速度 = 距离 / 时间)
a = Δv / Δt   (加速度 = 速度变化 / 时间)
F = m × a   (力 = 质量 × 加速度)
W = F × d   (功 = 力 × 距离)
P = W / t   (功率 = 功 / 时间)
V = I × R   (欧姆定律：电压 = 电流 × 电阻)
P = V × I = I² × R   (电功率)
v = f × λ   (波速 = 频率 × 波长)
g = 9.8 m/s²   (重力加速度)

⚗️ 化学公式

M = n / V   (摩尔浓度 = 摩尔数 / 体积)
M₁V₁ = M₂V₂   (稀释公式)
pH = -log[H⁺]   (pH计算公式)
PV = nRT   (理想气体状态方程)
密度 = 质量 / 体积

🧬 生物/生态公式

种群增长率 = (出生数 - 死亡数) / 总数
能量传递效率 ≈ 10% (营养级之间)
光合效率 = 产生的O₂量 / 消耗的CO₂量

🌍 地球科学公式

地震波速 = 距离 / 时间
密度 = 质量 / 体积 (用于矿物鉴定)
1光年 ≈ 9.46 × 10¹² km
